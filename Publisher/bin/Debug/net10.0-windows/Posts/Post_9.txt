<b>Sandbox برای Agentهای کدنویس</b>

Agent موفق فقط مجموعه‌ای از LLM callها با prompt و context نیست. الگوی قوی‌تر زمانی شکل می‌گیرد که Agent داخل Sandbox بتواند code بنویسد، Toolهای خودش را در لحظه بسازد و حتی sub-agentهایی ایجاد کند که کار او را evaluate کنند. این ساختار باعث می‌شود Agent به‌جای پاسخ متنی محدود، وارد محیط عملی شود و خروجی خود را بسازد، بررسی کند و تکرار کند. چنین مدلی برای بستن فاصله آخر در کارهای پیچیده موفق‌تر است.
