<b>تفکیک Frontend، Runtime و Ingestion Layer</b>

معماری Agentهای مقیاس‌پذیر به سه بخش اصلی تقسیم می‌شود: لایه ingestion برای دریافت رویدادها، لایه runtime برای اجرای Agent و لایه frontend برای تعامل کاربر. این تفکیک باعث می‌شود هر بخش به‌صورت مستقل مقیاس‌پذیر باشد و خطاها به‌صورت ایزوله مدیریت شوند. همچنین ارتباط بین این لایه‌ها معمولاً از طریق API و Pub/Sub انجام می‌شود.
