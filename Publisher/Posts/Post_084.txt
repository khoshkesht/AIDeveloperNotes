#Post_2 <b>راه‌اندازی اولیه Hermes Agent</b>

راه‌اندازی Hermes Agent از طریق Desktop App یا Terminal انجام می‌شود و پس از نصب، اولین مرحله انتخاب Model Provider مناسب است. امکان اتصال به Providerهای مختلف یا استفاده از Modelهای محلی وجود دارد و پس از برقراری اتصال، Agent آماده شروع تعامل خواهد بود. یکی از اولین اقداماتی که پیشنهاد می‌شود، معرفی اطلاعات اولیه کاربر و درخواست ذخیره آن در Memory است تا Agent از همان ابتدای کار شناخت اولیه‌ای از کاربر داشته باشد.
