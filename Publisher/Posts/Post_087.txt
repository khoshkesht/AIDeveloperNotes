#Post_5 <b>ساختار داخلی Memory در Hermes</b>

اطلاعات اصلی Hermes در پوشه `.hermes` ذخیره می‌شود و بخش مهمی از Memory به صورت فایل‌های Markdown قابل مشاهده است. اطلاعات مربوط به کاربر، تنظیمات کلی و هویت Agent در فایل‌های جداگانه نگهداری می‌شوند و همین موضوع باعث می‌شود Memory علاوه بر استفاده توسط Agent، برای توسعه‌دهنده نیز قابل بررسی و درک باشد. این ساختار امکان مشاهده مستقیم داده‌های ذخیره‌شده را فراهم می‌کند.
