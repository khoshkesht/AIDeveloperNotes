#Post_1 <b>انتخاب محل مناسب برای اجرای Hermes Agent</b>

انتخاب محیط اجرای Hermes Agent یکی از تصمیم‌های اولیه‌ای است که بر عملکرد، هزینه و شیوه استفاده از Agent تأثیر می‌گذارد. اجرای دائمی روی یک ماشین محلی قدرتمند، VPS، یک لپ‌تاپ قدیمی یا حتی کامپیوتر شخصی امکان‌پذیر است و هرکدام مزایا و محدودیت‌های خود را دارند. اگر Agent روی سیستم شخصی اجرا شود، استفاده از Docker برای جداسازی محیط اجرا توصیه می‌شود تا فعالیت Agent از سایر بخش‌های سیستم مستقل باقی بماند.
