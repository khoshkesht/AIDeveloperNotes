<b>Context پویا با Agent Skills</b>

وقتی context سنگین می‌شود، بارگذاری کامل اطلاعات همیشه راه‌حل خوبی نیست. Agent Skills کمک می‌کند context فقط در زمان نیاز به‌صورت dynamic وارد کار شود، نه اینکه Agent از ابتدا همه اطلاعات را همراه داشته باشد. این روش برای code baseهای بزرگ اهمیت زیادی دارد، چون Agent می‌تواند در لحظه به بخش مناسب context دسترسی پیدا کند. نتیجه، کاهش فشار context و افزایش شانس رسیدن به خروجی قابل‌اعتماد در وظایف پیچیده‌تر است.
