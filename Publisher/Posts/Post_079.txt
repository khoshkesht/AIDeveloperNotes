<b>Agent UI Layer روی Runtime</b>

برای بهبود تجربه کاربری، یک UI layer روی Agent runtime قرار می‌گیرد. این لایه امکان تعامل غیرمستقیم با Agent را فراهم می‌کند و به جای chat ساده، از یک رابط گرافیکی استفاده می‌شود. ارتباط بین UI و backend از طریق session service و API gateway انجام می‌شود تا تمامی درخواست‌ها به‌صورت ساختارمند به runtime منتقل شوند.
